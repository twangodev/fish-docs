## Available Models

Currently available models:
- **OpenAudio S1** (Recommended) - Latest generation with best performance
- **speech-1.6** - Previous generation, stable for existing integrations
- **speech-1.5** - Earlier generation, maintained for compatibility

<Note>
We strongly recommend using **OpenAudio S1** for all new projects to access the latest capabilities and performance improvements.
</Note>
