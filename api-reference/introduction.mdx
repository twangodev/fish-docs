## Welcome

You can generate a new API key at [https://fish.audio/app/api-keys/](https://fish.audio/app/api-keys/).

## Quick Start
See our [Quick Start](/developer-platform/getting-started/quickstart) guide to generate audio in under 2 minutes. 

## Create a Voice Clone
Use our [/model endpoint](/api-reference/endpoint/model/create-model) to create a voice clone model.

## Generate Speech
Use our [/v1/tts endpoint](/api-reference/endpoint/openapi-v1/text-to-speech) to generate speech.

## Realtime Streaming
Use our [SDK](/sdk-reference/python/websocket) for realtime audio streaming with WebSocket. 

## Rate Limits

You can find the rate limits for each endpoint in the [Rate Limits](/developer-platform/models-pricing/pricing-and-rate-limits) section.
